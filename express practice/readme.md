## Practice of Express js

**1. npm init -y**

- package.json file was created

**2. npm i express**

```json
"dependencies": {
    "express": "^4.21.1"
  },
```

**3. npm i --save-dev nodemon**

```json
"devDependencies": {
    "nodemon": "^3.1.7"
  }
```

**4. in package.json file**

```json
"scripts": {
 "start": "nodemon server.js",
}
```
